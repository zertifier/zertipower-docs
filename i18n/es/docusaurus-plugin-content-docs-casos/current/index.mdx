# Casos de Uso

**Explorando la Inteligencia Regenerativa en Acción**

Esta sección recopila aplicaciones del mundo real de la tecnología RegenAg-X. Hemos estructurado los casos en dos categorías principales para facilitar la navegación:

*   **Zertifarm (Suelo):** Casos centrados en la salud del suelo, análisis de nutrientes y prácticas regenerativas en granjas piloto.
*   **Zertiair (CO2):** Casos centrados en la medición y análisis de CO2 al aire libre como indicador de calidad ambiental.

Selecciona una de las tarjetas a continuación para profundizar en los detalles, conjuntos de datos y algoritmos específicos de cada dominio.
