# Anàlisi de CO2 (Zertiair)

Explora els recursos de Zertiair per al monitoratge i anàlisi de CO2. Aquesta secció ofereix accés tant a datasets crus per a recerca independent com a algoritmes avançats de Compute-to-Data que permeten extreure intel·ligència ambiental—des de patrons temporals i regionals fins a la detecció d'anomalies—sense comprometre la privacitat de les dades.
