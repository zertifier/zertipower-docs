# Ocean Protocol, mercat i IPFS

Ocean Protocol habilita mercats de dades i de serveis d'IA amb contractes intel·ligents i un sistema de permisos flexible. Els actius (datasets, algoritmes) es publiquen amb metadades, polítiques i preus; les referències als fitxers poden residir a IPFS o orígens privats.

Components clau
- Tokens i transaccions: cada actiu pot tokenitzar‑se (ERC‑20) per a governança i preus; el consum es paga per transacció on‑chain.
- Ocean Node: orquestra publicació, permisos, Compute‑to‑Data i registre en blockchain.
- Sistema distribuït: les dades romanen on són mentre el còmput es desplaça cap a la dada.

Avantatges
- Monetització transparent i auditables.
- Separació entre metadades públiques i dades privades.
- Integració amb Compute‑to‑Data per protegir la sobirania de la dada.
