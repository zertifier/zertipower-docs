# Use Cases

**Exploring Regenerative Intelligence in Action**

This section gathers real-world applications of RegenAg-X technology. We have structured the cases into two main categories to facilitate navigation:

*   **Zertifarm (Soil):** Cases focused on soil health, nutrient analysis, and regenerative practices in pilot farms.
*   **Zertiair (CO2):** Cases focused on outdoor CO2 measurement and analysis as an environmental quality indicator.
